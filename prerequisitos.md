# Linux

En mucho más fácil que en windows

## Distro Debian y sus derivados

```bash
sudo apt update
sudo apt install build-essential -y
```

## Distro fedora y sus derivados

```bash
sudo dnf groupinstall "Development Tools"
# O también 
sudo dnf install gcc-c++
```

## Distro Arch y sus derivados

```bash
sudo pacman -S base-devel
```
