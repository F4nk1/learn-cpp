# Learn C++

Repositorio con un conjunto de herramientas y lecciones diseñadas para aprender y practicar **C++** de forma progresiva.

---

## Requisitos

- Estándar de C++: **C++17**
- Compilador compatible (g++)
- `make`

---

## 🚀 Compilación y ejecución

Este proyecto utiliza un **Makefile** para simplificar la compilación y ejecución de los programas.

### ▶️ Comando general

```bash
make run <nombre_problema>
